# Design Companion Placeholder

This directory is reserved for Pencil.dev and other companion design artifacts that
support the shared system workflow.

Important:

- Code remains the source of truth.
- Pencil boards are the preferred visual review surface for major shared design work.
- No approved token, component, or interaction rule should live only in this folder.
- If a design file introduces a new pattern, mirror it back into code, docs, and agent
  rules before treating it as canonical.
- Track active boards in `board-index.md`.
- Use `handoff-checklist.md` before treating a reviewed board as implementation-ready.
